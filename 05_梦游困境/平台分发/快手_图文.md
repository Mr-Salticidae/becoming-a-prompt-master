## 正文

群里每天有个 prompt battle，今天的题是「梦游困境」。

这题俩词，多数人只做了「梦游」——画个闭眼走路的人，全挤在"走向悬崖边"，撞成一团。

但"困境"才是题眼。我把它做成主体：睡衣人踏出白楼梯最后一级，脚下万米云海城市，身后楼梯还在一节节消失——前面坠落，后面没路，他却没醒，还往前迈。这才是"困"。

两条挺有用的经验：

①「困境」要做成"进退两难"，不是"走向危险"。加一句"身后路也断了"，那个"困"才立得住。

②嫌"太暗、没美感"，别在原图上调亮——整组换风格词。我把冷暗惊悚那套全删，换成复古未来（库布里克《2001》+粉彩），一版就成，画面还更高级。

完整 prompt 和四层拆解都在图里，可以直接抄去试。

第 5 期，《目标是成为 Prompt 大师》。

## 发布清单（对齐快手 SOP 七步）

- **Step1 入场票审计**：视觉奇观（踏空云海冠军图）+ 系列识别符（目标是成为 Prompt 大师，自建 IP）+ meme 标题。前 3 秒可识别 ✓，AI 教程类在快手有受众 → **审计通过，可同步主推**。
- **Step2 适配层**：只动封面/标题/标签，作品内核与拆解不动。
- **Step3 时机**：20:30–21:30 发布（晚高峰前段，留 4h 观察窗）；与抖音错开 ≥30 分钟。
- **Step4 24h 静默**：只观察，看完播率 / 点赞率（最可靠）/ 评论质量 / 涨粉转化，不看绝对播放量。
- **Step5 双轨诊断**：播放低+点赞率>3% → 改分发不投流；播放正常+点赞率<0.3% → 改封面/前 3 秒。
- **投流**：只投自然流量已验证的；点赞率异常高+涨粉成本低才追投，不买虚假信号。
