# 目标是成为 Prompt 大师

一个面向 AIGC 小白的 prompt 拆解系列。

每一期挑一个 prompt battle 主题,讲清楚四件事:
**难在哪 → 怎么破题 → 完整 prompt → 四层拆解与可复用的思路。**

每篇笔记都可**单独阅读**,不依赖其它文档;配套有可直接发布的小红书卡片(3:4 竖版)和小红书正文。

---

## 目录

| 期数 | 主题 | 分类 | 一句话 | 文件夹 |
|---|---|---|---|---|
| 第 1 期 | 《对话》 | 抽象词破题 | 不画两个人说话,让两个时代做同一个动作 | [01_对话](01_对话/) |
| 第 2 期 | 《童话镇里的纸飞机》 | 物件与尺度 | 不让纸飞机飞,让它落地变成一座可攀爬的山 | [02_纸飞机](02_纸飞机/) |

> 网站按「分类」组织:首页是封面墙(可按分类筛选),点开任意一期进入详情。新增一期 = 加一个文件夹 + 在 `index.html` 的 `EPISODES` 数组里加一条。

---

## 每期文件夹里有什么

- `XX_笔记.md` —— 独立可读的拆解笔记(小白向,长 · 存档)
- `XX_小红书正文.md` —— 可直接发布的小红书正文(短 · 内敛克制)
- `XX_封面卡.png` —— 小红书封面图
- `XX_prompt拆解卡.png` —— prompt 四层标注图
- `XX_原图.png` —— 作品原图
